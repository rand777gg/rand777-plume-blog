Apache HTTP 服务器（通常简称为 Apache）是一个开源的跨平台 Web 服务器软件，由 Apache 软件基金会维护。它是世界上使用最广泛的 Web 服务器之一，支持多种操作系统，包括 Linux、Windows 和 macOS。
